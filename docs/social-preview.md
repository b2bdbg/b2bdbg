# Social preview image

GitHub shows a 1280×640 px social preview image when links to this repository
are shared on social networks, Slack, or messaging apps.

## Recommended content

A dark background with:
- Project name: `B2BDBG`
- Tag line: `trace and debug bot-to-bot conversations`
- A simplified version of the ASCII architecture diagram from README.md
  (proxy box → Jaeger + Prometheus + Grafana)
- The three technology logos: Go gopher, OpenTelemetry, Prometheus (check
  respective brand guidelines for permitted usage)

## Current image

`docs/img/preview.png` is the project preview/screenshot (also used as the
README hero). Use it as the GitHub social preview until a dedicated 1280×640
card is produced.

## How to set it

1. Use `docs/img/preview.png` (or a dedicated 1280×640 PNG/JPEG).
2. Go to **Settings → Social preview → Edit**.
3. Upload the image.

GitHub stores the social preview in repository settings (not in the tree);
`docs/img/preview.png` is committed so the README hero renders.
