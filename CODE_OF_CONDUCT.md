# Code of conduct

This project follows the [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

In short: be respectful, assume good faith, and focus on the work.
Harassment, personal attacks, and discriminatory language are not acceptable
in issues, pull requests, discussions, or any other project space.

Violations may be reported privately to the maintainers via
[GitHub private vulnerability reporting](https://github.com/b2bdbg/b2bdbg/security/advisories/new)
or by opening a minimal non-sensitive issue requesting a private channel.
